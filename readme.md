Chat build in nodejs, express, socket.io, mongo, graphql, and backbone

for client development
  - cd client
  - yarn install
  - yarn start

for server development run node on por 3000
  - cd server
  - yarn install
  - yarn start

for server testing, it uses mocha, supertest and chai

  - yarn watch
